| Nombre  | Regla de nombrado  | Ubicación física |

    * \<nombre-metrica\> = nombre en letras minusculas, espacio reemplazado por guión medio (-).
    * \<nombre-prueba\> = nombre en letras minusculas, espacio reemplazado por guión medio (-).
